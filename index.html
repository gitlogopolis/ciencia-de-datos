Hola. Python está viviendo una segunda juventud como lenguaje de programación alineado codo con codo con una de las grandes necesidades de esta década: el trabajo con grandes volúmenes de datos. En este curso vas a aprender, en primer lugar, los fundamentos de Python para el "data science" y visualización de datos, para posteriormente aplicarlos en el análisis de grandes volúmenes de datos usando el entorno Spark, una de las herramientas más populares del sector "big data". El enfoque del contenido de este curso es eminentemente práctico, haciendo especial énfasis en sentar las bases para que aprendas a desarrollar tu propio proyecto de datos de principio a fin. Mi nombre es Joan Gasull, soy científico de datos, así como profesor de estadística y programación, y en este curso voy a acompañarte en tus primeros pasos como científico de datos con Python e iniciarte en una de las disciplinas más demandadas hoy en día en el mercado laboral. ¿Empezamos?
